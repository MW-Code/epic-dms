## Aufnahme-Hinweise

- Browser-Fenster bei mind. 1280px Breite, sonst kollabiert das 3-Spalten-Layout
- Light- oder Dark-Mode konsistent halten (alle 4 Bilder gleich)
- PNG bevorzugt, JPG ist Plan B
- Datei-Groesse: idealerweise unter 500 KB pro Bild — sonst ladet GitHub das README langsam
- Sensible Daten ausblenden / mit Test-Dokumenten arbeiten
